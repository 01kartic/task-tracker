# Task Tracker

A modern desktop application for tracking daily tasks and building consistent habits. Task Tracker helps you create custom trackers with multiple tasks to complete daily, mark them as complete, incomplete, or skipped each day, and monitor your progress with streak counters to stay motivated.

![App Preview](/public/app-preview.png)

View your productivity with interactive charts and statistics in the analytics dashboard. The app features dark/light mode support, desktop notifications for reminders, and stores all your data locally using IndexedDB for complete privacy. Available for macOS. Windows will released soon.
